# 使用Rust编写操作系统

使用Rust编写操作系统，屏蔽掉过于底层的硬件细节和汇编语言，这有助于将主要精力放在内存管理、进程管理、文件系统等核心模块。

对操作系统启动之前的主引导，跳入保护模式，加载内核等过程有兴趣的可以参考[reduced-os（已封存）](https://github.com/niguangye/reduced-os)
